# Technology Stack

## Core Technologies

- **HTML5**: Single-page application structure
- **Vanilla JavaScript**: All application logic, no frameworks
- **CSS3**: Custom styling with CSS variables
- **Tailwind CSS**: Utility-first CSS framework (CDN)
- **localStorage API**: Client-side data persistence

## External Dependencies

- **Tailwind CSS CDN**: `https://cdn.tailwindcss.com`
- **Google Fonts**: Cairo and Tajawal Arabic font families

## Architecture

- Single HTML file (`index.html`) containing all markup, styles, and logic
- No build system or compilation required
- No backend or server-side components
- Pure client-side application

## Running the Application

Simply open `index.html` in a modern web browser. No build steps, installation, or server required.

## Browser Requirements

- Modern browser with ES6+ JavaScript support
- localStorage API support
- SVG rendering support
