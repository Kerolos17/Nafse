# Product Overview

نافس (Nafis) is a 90-day habit tracking and learning journey application designed for Arabic-speaking developers learning WordPress development. The app helps users track daily progress through three 30-day phases: Fundamentals, Development, and Mastery.

## Core Features

- Personal goal setting and journey tracking
- Daily habit/task management with completion tracking
- 90-day calendar visualization with streak tracking
- Three-phase learning structure (30 days each)
- Study timer (Pomodoro-style)
- Daily mood tracking and journaling
- Progress analytics and statistics
- Local-first data storage (localStorage)

## Target Audience

Arabic-speaking developers pursuing WordPress development skills through a structured 90-day program combining study, practical work, and fitness habits.
